## 개발 환경 구성

1. 프로젝트 다운로드
    ```
    git clone <repo URL> <folder-name>
    ```

2. frontend 폴더로 이동
    ```
    cd <folder-name>/frontend
    ```

3. 패키지 설치
    ```
    npm install
    ```

4. 프로젝트 실행
    ```
    npm run serve
    ```


## 디렉토리 구조

```
.
└─src
    ├─api
    ├─assets  /* image, css, js 등의 리소스 */
    ├─components  /* 컴포넌트 단위의 Vue 파일 */
    ├─views  /* 페이지 단위의 Vue 파일 */
    └─vuex  /* Vuex 관련 파일 */
```
